重置密码

  1. 关闭MySQL服务
    /usr/local/mysql/support-files/mysql.server
  2. 切换目录
    cd /usr/local/mysql/bin
  3. 获取权限
    sudo su
  4. 重启服务
    ./mysqld_safe --skip-grant-tables
  5. 重开终端
    alias mysql=/usr/local/mysql/bin/mysql
  6. 进入MySQL命令模式
    mysql
  7. 选择MySQL数据库
    use MySQL
  8. 刷新权限
    flush privileges
  9. 重置密码
    set password for ‘root’@’localhost’=password(‘新密码’)

常用操作
